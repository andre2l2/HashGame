# Hash Game

## Como rodar em sua maquina:

* Faça um clone deste repositorio para sua maquina;
* Execute os comndos abaixo;

```terminal
    % npm install 
    % npm run start    
```

Após isso você poderá abrir em: http://localhost:3333
